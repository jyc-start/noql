# Mathematical Functions

## `#!sql AVG`

`#!sql AVG(expr,expr,...)`

Returns the average of a set of values.


???+ example "Example `#!sql AVG` usage"

    ```sql
    SELECT
        AVG(`Replacement Cost`,`id`) AS exprVal 
    FROM 
        `films` 
    ```
